# estudo-notebooklm
Projeto inicial de contato com o NotebookLM.

Este projeto consiste na construção de uma base de conhecimento utilizando o Google NotebookLM para centralizar informações relacionadas ao pronunciamento contábil CPC 06 (R2) – Operações de Arrendamento Mercantil, convergente à norma internacional IFRS 16.

O objetivo é disponibilizar uma ferramenta de consulta rápida para profissionais das áreas Financeira, Contábil e Controladoria, permitindo acesso simplificado às principais definições, critérios de reconhecimento, mensuração, contabilização e divulgação exigidos pela norma.

Objetivo do Projeto

Empresas frequentemente enfrentam dificuldades na disseminação e retenção de conhecimento técnico sobre pronunciamentos contábeis complexos.

Este projeto busca:

Consolidar conteúdos técnicos em uma única fonte de consulta.\
Reduzir o tempo gasto na busca por informações normativas.\
Apoiar colaboradores na resolução de dúvidas pontuais.\
Padronizar interpretações da norma dentro da organização.\
Demonstrar o uso de Inteligência Artificial Generativa aplicada à gestão do conhecimento corporativo.\
Problema de Negócio

A interpretação do CPC 06 (R2) exige consulta frequente a documentos extensos e técnicos.

Entre os desafios observados estão:

Dificuldade de localizar rapidamente tópicos específicos.\
Divergências de interpretação entre usuários.\
Dependência de especialistas para esclarecimento de dúvidas recorrentes.\
Baixa centralização das informações normativas.\
Solução Proposta

Foi desenvolvido um NotebookLM contendo uma base documental estruturada sobre o CPC 06 (R2), permitindo que usuários realizem consultas em linguagem natural.

Exemplos de perguntas suportadas:

O que caracteriza um arrendamento segundo o CPC 06?\
Como calcular o passivo de arrendamento?\
Quais são as exceções de reconhecimento?\
Qual a diferença entre arrendamento operacional e financeiro sob a norma anterior?\
Como ocorre a contabilização inicial do ativo de direito de uso?\
Fontes Utilizadas\
Fontes Primárias\
Pronunciamento Técnico CPC 06 (R2)\
IFRS 16 – Leases\
Material oficial do Comitê de Pronunciamentos Contábeis (CPC)\
Exemplos ilustrativos da IFRS Foundation\
Fontes Complementares\
Artigos técnicos de auditorias independentes\
Guias de implementação\
Estudos de caso e exemplos práticos

Metodologia\
Coleta de materiais oficiais.\
Seleção de documentos relevantes.\
Upload das fontes para o NotebookLM.\
Testes de perguntas e validação das respostas.\
Ajustes da base documental.\
Elaboração de guia de utilização para usuários finais.
